# Laravel com Kinde

- Projeto de autenticação social utilizando o provedor do kinde.
- Template com Laravel + Inertia + TailwindCSS + Shadcn UI

### Requisitos
- Php
- Node.js
- Projeto no Kinde

### Preparação
- Criar arquivo **.env** e preencher as informação do Kinde.

### Execução
1. ```composer install```
2. ```npm install```
3. ```composer migrate``` 
4. ```composer dev```
